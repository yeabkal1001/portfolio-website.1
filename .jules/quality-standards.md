# Awwwards Quality Standards

To achieve Awwwards-quality results, the following standards must be met by all agents:

## Performance & Optimization
- **Lighthouse Performance Score**: 90+
- **First Contentful Paint (FCP)**: < 1.0s
- **Time to Interactive (TTI)**: < 2.5s
- **60fps Animations**: Use GSAP and Framer Motion efficiently to ensure no dropped frames.
- **Image Optimization**: Use Next/Image for all assets.

## Design & UX
- **Mobile-First**: Design and develop for mobile responsiveness from the start.
- **Accessibility (WCAG 2.1 AA)**:
  - Proper contrast ratios.
  - Semantic HTML5.
  - Keyboard navigability.
  - Screen reader support.
- **Visual Fidelity**: Pixel-perfect implementation of the design system (Deep Dark Mode, Neon Green accents).

## Technical Excellence
- **TypeScript Throughout**: No `any` types. Strict type checking enabled.
- **Clean Code**: Follow component-based architecture. DRY principles.
- **SEO**: 90+ Lighthouse SEO score. Proper meta tags and structured data.

## Animation & Interaction
- Smooth scrolling (Lenis Scroll).
- Subtle WebGL/3D effects where appropriate.
- Custom cursor with interactive states.
